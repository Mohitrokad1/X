Help
  • Click to upload image
  • Click to draw and crop
  • Ctrl + click on stroke to add a point
  • Shift + click on point to remove it
  • Ctrl + Shift + click on the drawing object to remove it
  • Drag to move selected object (background,sharp and points)
  • Mouse wheel to zoom in/out selected object
  • Click to get all positions and cropped images
  • Click to get clear all drawing
